# TENSclic 🏥

**TENSclic** es una solución tecnológica desarrollada en Python para conectar a Técnicos en Enfermería (TENS) con cuidadores y pacientes, optimizando la gestión de servicios de salud domiciliaria.

Este proyecto fue diseñado con un enfoque especial en la comunidad de **San Bernardo**, integrando servicios de geolocalización y respuesta ante emergencias.

## 🚀 Características principales
*   **Arquitectura Modular**: Código organizado en servicios independientes (`api_services.py`, `geolocalizacion.py`).
*   **Gestión de Base de Datos**: Uso de SQLite para el registro seguro de pacientes y profesionales.
*   **Geolocalización**: Integración con servicios de mapas para ubicación de servicios de emergencia.
*   **Interfaz Moderna**: Desarrollada con la librería **Flet**.

## 🛠️ Tecnologías utilizadas
*   **Lenguaje**: Python 3.x
*   **GUI**: Flet (basado en Flutter)
*   **Base de Datos**: SQLite
*   **Versionamiento**: Git & GitHub
